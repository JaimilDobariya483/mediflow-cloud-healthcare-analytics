MediFlow Cloud — Healthcare Readmission Analytics
Project Overview

MediFlow Cloud is a cloud-native healthcare analytics platform designed to analyze hospital readmission trends and optimize healthcare resource utilization. This project uses AWS services to build a scalable data pipeline and Power BI for interactive dashboard visualization.

The goal of this project is to help hospitals:

Reduce patient readmission rates
Optimize hospital resource allocation
Identify high-risk patient groups
Improve healthcare decision-making
Architecture

The project follows a cloud-native architecture:

Data Sources → AWS S3 (Raw Data) → AWS Athena (Query Engine) → Processed Data → Power BI Dashboard

Data Pipeline
Step 1: Data Generation

Synthetic healthcare dataset generated using Python and Pandas.

Step 2: Data Storage

Data uploaded to AWS S3 Raw Zone.

Step 3: Data Processing

AWS Athena used for querying and transformation.

Step 4: Data Export

Processed datasets exported for dashboard visualization.

Step 5: Dashboard Creation

Power BI dashboard built for analytics and reporting.
